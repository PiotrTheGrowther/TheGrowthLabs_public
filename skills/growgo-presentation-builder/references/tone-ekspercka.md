# Growgo Tone - Ekspercka

Tone of voice rules for Growgo decks. This is the default mode for presentations, reports, and offers (per Growgo brand router).

## Character

Growgo speaks as **doradca-analityk** (advisor-analyst) - not as Jan's personal voice. Confident, expert, backed by data and experience. No personal opinions, no humor, no asides. Professional, but no empty corporate jargon.

## Core principle: advisor, not decision-maker

Growgo **does not recommend, advise, or instruct**. Growgo proposes ideas, shows options, points to benefits - the decision belongs to the client. This is a legal-responsibility rule.

| FORBIDDEN (directive) | ALLOWED (firm but non-directive) |
|---|---|
| "Rekomendujemy wdrożenie X" | "Proponujemy wdrożenie X" |
| "Zalecamy zmianę podejścia" | "Warto zmienić podejście na..." |
| "Należy wdrożyć Y" | "Wdrożenie Y przyniesie..." |
| "Trzeba zmienić Z" | "Zmiana Z może przynieść istotne korzyści" |
| "Konieczne jest..." | "Silną opcją jest..." / "Priorytetem powinno być..." |
| "Musicie zrobić X" | "Najskuteczniejszym podejściem jest X" |
| "Nasza rekomendacja to..." | "Na podstawie analizy, proponujemy X" |

English equivalents:
| FORBIDDEN | ALLOWED |
|---|---|
| "We recommend X" | "We propose X" |
| "You should do Y" | "Y delivers..." / "The key benefit of Y is..." |
| "It's necessary to..." | "A strong option is..." / "The priority is..." |
| "You must X" | "The most effective approach is X" |
| "Our recommendation is..." | "Based on the analysis, we propose..." |

## No personal narration

In expert mode: never write "I think", "I see", "for me", "I was surprised". Write from analysis + data perspective.

| FORBIDDEN | ALLOWED |
|---|---|
| "Myślę że..." / "I think that..." | "Analiza wskazuje..." / "Analysis indicates..." |
| "Widzę że..." / "I see that..." | "Zidentyfikowano..." / "We identified..." |
| "Dla mnie kluczowe było Y" | "Kluczowym elementem jest Y" |
| "Zaskoczyło mnie, że X" | "Wyniki wskazują, że X" |

## Formatting

- **Bullet points and numbered lists** are standard. 3+ items - use a list.
- Hierarchical structures (headings + subpoints) preferred.
- Each bullet concise and concrete.
- Tables for comparisons and data.
- **Zero asides, zero humor, zero loose parenthetical remarks.**
- **Zero emoji.** Anywhere.

## Presentation-specific (this skill)

- **Tight and energetic** - every word earns its place
- Short phrases, not full sentences (in titles)
- Active voice, benefit-led language
- Hook headlines, not labels ("Co zyskasz" / "What you'll gain", not "Agenda")
- Data and specifics, not generalities

## Forbidden universal phrases (Growgo brand)

These are banned across all Growgo communication:

| Instead of | Write |
|---|---|
| "Mieliśmy ogromną przyjemność" | "Pracowaliśmy z X" / "Zrealizowano projekt z X" |
| "Z przyjemnością podzielimy się" | "Chętnie opowiem więcej" / "Szczegóły w dalszej części" |
| "W zakresie" | "o", "na temat" |
| "Niezwykle ważny" | drop or replace with concrete fact |
| "Pragniemy zaprezentować" | "Pokażemy" / "Przedstawiono" |
| "Mechanizm funkcjonowania polega na" | "To działa tak" / "Proces przebiega następująco" |

Forbidden patterns:
- Corporate jargon (empty phrases that sound professional but say nothing)
- Guru tone (lecturing from a position of superiority)
- Hype (promises without backing - "AI will do everything for you")
- Scaremongering ("Who doesn't implement AI, will die")
- Over-simplification ("It's simple, 5 minutes and done")
- Direct self-promotion ("At our Growgo we...", "In our trainings...")
- **Emoji** (🔥💡🚀🎯) - NEVER

## Language convention

- **Polish source = Polish output.**
- **English source = English output.**
- Polish typographic quotes: „..." (low open, high close) - never "..." or "..."
- Polish thousands separator: space (`6 000+`)
- Polish currency: `12 450,00 zł`
- English thousands separator: comma (`6,000+`)
- English currency: `$12,450.00`
- Polish dates: `19.05.2026`
- English dates: `2026-05-19`

## Brand spelling

Always **Growgo** - never "GrowGo", "GROWGO", or "growgo" (lowercase only in URLs/handles).

## Verification checklist

Before finishing the deck:

1. **Recommendation test**: Anywhere I recommended, advised, or instructed? Replace with "proponujemy" / "the key benefit is".
2. **Personal narration test**: Any "I think", "I see", "for me"? Replace with impersonal.
3. **Gravity test**: Any humor, asides, loose remarks? Remove.
4. **Brand spelling test**: Always "Growgo"?
5. **Emoji test**: Any emoji anywhere? Remove.
6. **Empty phrase test**: Any of the forbidden universal phrases? Replace.
7. **Concrete test**: Each claim backed by example, data, or experience? If not, remove.
