04 / Get it running

Four moves and you are generating.

  1. 01

    Download the skill zip.

    Grab the Presentation Builder bundle from the Substack link below.

  2. 02

    Drag it into Settings - Capabilities.

    Cowork unpacks it. The skill becomes invocable in every new task.

  3. 03

    Open a folder with your source content.

    A transcript, a newsletter, a PDF, a Notion export. Anything written.

  4. 04

    Tell Claude to use the skill.

    "Use Presentation Builder on the transcript in this folder." That is the whole prompt.

05 / What is inside

The skill is a folder, not a black box.

Every rule that drives the output is a markdown file you can read and edit. Slide types, design system, building protocol, verification rules. Open it up and tune it.

09 / What runs in the background

Three minutes, broken down.

  1. 0:00 - 0:20

    Read the skill, locate the transcript.

    Glob the folder, pick the right file, load the building protocol.

  2. 0:20 - 1:00

    Chunk content, map to slide types.

    Hook, metric, before-after, quote, steps. 8 to 14 sections in the outline.

  3. 1:00 - 1:30

    Pick an aesthetic, commit to it.

    For Growgo variant, this step is skipped. Aesthetic is locked.

  4. 1:30 - 2:40

    Assemble the single HTML file.

    Base scaffold plus section partials plus your copy. CSS and JS inlined.

  5. 2:40 - 3:00

    Verify. Open in browser. Hand back.

    Screenshot the hook, scan for layout breaks, fix any issues found.
